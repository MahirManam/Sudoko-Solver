# Sudoku Solver 🧩

A web-based Sudoku Solver built using HTML, CSS, and JavaScript.  
An interactive web-based tool that solves Sudoku puzzles using a backtracking algorithm, allowing users to input a puzzle and instantly generate the correct solution.

## 🚀 Features
- Interactive 9x9 Sudoku grid
- Automatic Sudoku solver
- Clean and responsive UI
- Error handling for invalid inputs

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
